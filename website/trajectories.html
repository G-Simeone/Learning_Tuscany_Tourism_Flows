Trajectory Clustering Methodology

We clustered tourist trajectories by analysing the sequences of municipalities visited by individuals. This analysis finds the most frequent paths taken by tourists. For tourists of a set of different nationalities who visited Tuscany in different seasons, we cluster the trips they had taken in Italy based on the individual spatial trajectories, or sequences, categorised by the set of municipalities they visited (e.g., Florence -> Pisa -> Livorno).

We used a Longest Common Subsequence based metric to compute the distance between trajectories, from which we compute a dissimilarity matrix between individuals. This is then clustered using the k-medoids algorithm to identify different clusters and a representative trajectories for each. This methodology takes into account the order (i.e., where did tourists come from and where do they go next), the timing (i.e., when does a tourist visit a municipality), and the duration of visit at each municipality for a given itinerary.
